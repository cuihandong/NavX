# NavX

The first navigation to stripling.

# Architecture

webpack
Materialize
vue.js
jQuery

# USAGE

```
// first
clone the RP

// second
npm install

// third
use webpack
```

# Structure

```
- node_modules
- public
 - bundle.js  // webpack output
 - static     // font, img
- src
 - main.js    // webpack entry
 - others
- index.html
- package.json
- webpack.config.js
```
